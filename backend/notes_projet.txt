MySQL (install and config):

>> sudo apt-get update
>> sudo apt-get install python3-dev
>> sudo apt-get install python3-dev libmysqlclient-dev
>> pip install mysqlclient
>> sudo apt-get install mysql-server --fix-missing
>> systemctl status mysql.service
PS: To restart server >> service mysql restart
>> mysql -u <user_name> -p

	Si message d'erreur de type '...user_name@localhost...':
	>> sudo mysql -u root
	mysql> USE mysql;
	mysql> CREATE USER 'YOUR_SYSTEM_USER'@'localhost' IDENTIFIED BY '';
	mysql> GRANT ALL PRIVILEGES ON *.* TO 'YOUR_SYSTEM_USER'@'localhost';
	mysql> UPDATE user SET plugin='auth_socket' WHERE User='YOUR_SYSTEM_USER';
	mysql> FLUSH PRIVILEGES;
	mysql> exit;
	>> service mysql restart
	>> mysql -u <YOUR_SYSTEM_USER>
	mysql> SHOW DATABASES;
	mysql> CREATE DATABASE game;
	mysql> SHOW DATABASES; (to check if OK)

Pour exporter les données de MySQL vers un fichier:
>> mysqldump --all-databases > dump.sql

Pour exporter une table spécifique:
>> mysqldump -u username -p database_name > dump.sql ("game" dans notre cas)

Pour importer la base de données depuis un fichier:
>> mysql -u username -p database_name < dump.sql
